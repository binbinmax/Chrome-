## 墙的介绍
 


    所谓的墙就是由原北京邮电大学校长设计（据说，是否真实不太清楚）一款长城防火墙，主要功能就是自动审查和过滤一些违规的信息，防火墙对于这些违规信息会用DNS污染（对于域名）和屏蔽IP的方式（对于IP）进行过滤，使国内用户无法访问，世界最大的搜索引擎谷歌就被DNS污染。
